## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
L'audit HTML a relevé l'absence de `<h1>` dans la page principale alors que le design system impose un seul `<h1>` par page et une hiérarchie de titres sans saut de niveau.

Revoir l'en-tête de page et les titres des sections/cartes pour réintroduire une hiérarchie sémantique correcte sans dégrader la présentation visuelle existante.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
- [ ] #1 La page contient exactement un `<h1>` visible ou sémantiquement correct
- [ ] #2 La hiérarchie des titres est cohérente entre l'en-tête de page, les sections et les dialogues
- [ ] #3 La modification n'altère pas le layout ni les styles attendus des cartes et headers
- [ ] #4 Les lecteurs d'écran bénéficient d'une structure de navigation plus claire
